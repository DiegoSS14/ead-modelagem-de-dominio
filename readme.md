# Plataforma EAD - Estudo de DDD

## Sobre

Projeto de estudo focado em **modelagem de domínio** com o objetivo de aplicar os princípios de **Domain-Driven Design (DDD)** em uma plataforma de Educação a Distância (EAD).

## Metodologia

O projeto é desenvolvido utilizando **Test-Driven Development (TDD)**, garantindo que o código seja guiado pelos testes desde o início.

## Estrutura

- **backend**: API desenvolvida em TypeScript com Jest para testes
- **frontend**: Interface da plataforma (em desenvolvimento)

## Como Executar

```bash
# Instalar dependências
npm install

# Rodar testes
npm test
```
